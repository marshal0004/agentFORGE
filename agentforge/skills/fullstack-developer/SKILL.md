# Fullstack Developer

Specialized multi-agent workflow for comprehensive full-stack development.

## Overview

The Fullstack Developer skill orchestrates multiple specialized agent personalities to handle every aspect of building web applications, from initial planning through deployment.

## Agent Personalities

This skill coordinates these specialized agents:

| Agent | Specialization | Color | Focus |
|-------|----------------|-------|-------|
| **Backend Architect** | System design | Purple | Scalability, architecture |
| **Code Reviewer** | Code quality | Orange | Security, best practices |
| **Database Optimizer** | Performance | Blue | Queries, indexing |
| **Frontend Developer** | UI/UX | Yellow | Design, implementation |
| **Rapid Prototyper** | Speed | Pink | Prototyping, iteration |
| **Senior Developer** | Premium quality | Green | Advanced techniques |
| **Reality Checker** | Testing | Red | Validation, bugs |
| **Test Automation Engineer** | Automation | Indigo | CI/CD, test suites |

## Workflow

### 1. Planning Phase
- Backend Architect designs scalable architecture
- Frontend Developer plans UI/UX approach
- Rapid Prototyper creates initial mockups
- Senior Developer identifies premium enhancement opportunities

### 2. Implementation Phase
- Backend Architect implements server-side logic
- Frontend Developer builds user interfaces
- Rapid Prototyper iterates on ideas
- Senior Developer applies premium design patterns

### 3. Quality Assurance Phase
- Reality Checker validates functionality
- Test Automation Engineer creates test suites
- Code Reviewer reviews all code
- Database Optimizer ensures performance

### 4. Deployment Phase
- Test Automation Engineer sets up CI/CD
- Backend Architect prepares production deployment
- Frontend Developer optimizes assets
- Senior Developer ensures premium experience

## Key Principles

### Collaboration
- Each agent contributes expertise
- Communication between agents ensures consistency
- Trade-offs are evaluated from multiple perspectives

### Quality
- Code quality, security, and performance are prioritized
- Testing is comprehensive and automated
- Design is premium and user-centered

### Speed
- Rapid prototyping accelerates development
- Automation handles repetitive tasks
- Iterative approach validates ideas quickly

## Technology Stack

### Backend
- Laravel, PHP
- Livewire, Alpine.js
- MySQL, PostgreSQL
- Redis, caching strategies

### Frontend
- React, Vue, or Alpine.js
- Tailwind CSS
- Three.js (for premium effects)
- FluxUI components

### DevOps
- CI/CD pipelines
- Docker containerization
- Automated testing
- Performance monitoring

## Success Criteria

- **Complete Application**: All requested features implemented
- **High Quality**: Clean code, secure, performant
- **User Experience**: Beautiful, intuitive, premium feel
- **Test Coverage**: Comprehensive tests with high pass rate
- **Documentation**: Clear, up-to-date documentation
- **Deployment**: Smooth deployment with monitoring

## Output Format

Each agent produces:
- Code with comments and documentation
- Test coverage reports
- Performance metrics
- Deployment guides
- Documentation updates

## Integration

This skill works best when combined with:
- Project management agents for task breakdown
- Design agents for visual specifications
- DevOps agents for deployment automation

## Notes

- Each agent has a distinct personality and expertise
- Agents communicate and coordinate to ensure quality
- The workflow is flexible and can be adapted to project needs
- All agents remember previous work and build on it
