# Fullstack Developer Skills

This directory contains specialized developer agent personalities for full-stack development workflows.

## Available Skills

### 🏗️ Backend Architect
**Specializes in**: System design, scalability, maintainability
- **Color**: Purple
- **Emoji**: 🏗️
- **Focus**: Scalable backend systems, Laravel/Livewire architecture, database design

### 🔍 Code Reviewer
**Specializes in**: Code quality, security, best practices
- **Color**: Orange
- **Emoji**: 🔍
- **Focus**: Pull request reviews, security analysis, performance optimization

### 🗄️ Database Optimizer
**Specializes in**: Query performance, database structure
- **Color**: Blue
- **Emoji**: 🗄️
- **Focus**: Query optimization, indexing strategies, database design

### 🎨 Frontend Developer
**Specializes in**: UI/UX implementation, responsive design
- **Color**: Yellow
- **Emoji**: 🎨
- **Focus**: Beautiful interfaces, Tailwind CSS, React/Vue patterns

### ⚡ Rapid Prototyper
**Specializes in**: Fast implementation, idea validation
- **Color**: Pink
- **Emoji**: ⚡
- **Focus**: Quick prototypes, iterative development, user feedback

### 💎 Senior Developer
**Specializes in**: Premium implementation, advanced techniques
- **Color**: Green
- **Emoji**: 💎
- **Focus**: Laravel/Livewire/FluxUI, Three.js integration, premium design

### ✅ Reality Checker
**Specializes in**: Testing, validation, bug detection
- **Color**: Red
- **Emoji**: ✅
- **Focus**: Unit tests, integration tests, edge case coverage

### 🤖 Test Automation Engineer
**Specializes in**: Automated testing, CI/CD pipelines
- **Color**: Indigo
- **Emoji**: 🤖
- **Focus**: Test suites, automation, continuous quality

## Usage

Each skill file contains:
- Agent personality and philosophy
- Critical rules and best practices
- Implementation process
- Technical expertise with code examples
- Success criteria
- Communication style
- Advanced capabilities

## Integration

These skills can be used in multi-agent workflows to:
1. **Plan** complex projects with multiple perspectives
2. **Implement** different aspects of a feature
3. **Review** code from various angles
4. **Test** thoroughly before deployment
5. **Optimize** performance and quality

## Notes

- Each skill has a distinct personality and focus
- Skills work together in a coordinated workflow
- Each skill has specific technical expertise and patterns
- Skills remember and build on previous work
